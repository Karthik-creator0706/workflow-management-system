# Workflow Management System UI

## Live Demo
Frontend (Versel): https://workflow-management-ui-six.vercel.app/
Frontend-Deploy(Render) : https://workflow-management-ui.onrender.com
Backend-Deploy(Render) : https://workflow-management-system-o317.onrender.com

## Project Overview
A Role-Based Workflow Management System built using React, Spring Boot, MySQL, and JWT Authentication.

## Tech Stack
- Spring Boot
- Spring Data JPA
- MySQL (Aiven)
- JWT Authentication

## Features
- User Login
- Role-Based Access Control
- Create Request
- View Requests
- Approve / Reject Requests
- Workflow Logs
- JWT Authentication

### Database
- MySQL (Aiven)

### Deployment
- Frontend: Render
- Backend: Render

## Roles
### USER
- Create Requests
- View Own Requests

### MANAGER
- Approve Requests
- Reject Requests
- Clarify Requests

### ADMIN
- Close Requests
- Reopen Requests
